# Bharath M N - Personal Portfolio

A modern, dark-themed personal portfolio website for Bharath M N, a BCA student.

## Features

- **Dark Theme**: Modern dark background with light grey and lavender gradients
- **Glassmorphism Design**: Beautiful glass-like cards with backdrop blur effects
- **Responsive Design**: Fully responsive and mobile-friendly
- **Smooth Animations**: Typing animation, hover effects, and smooth scrolling
- **Clean UI**: Minimal and professional layout
- **Modern Fonts**: Uses Inter font from Google Fonts

## Sections

1. **Hero**: Introduction with typing animation and call-to-action buttons
2. **About**: Personal information and profile placeholder
3. **Skills**: Technical skills displayed in glassmorphism cards
4. **Projects**: Project showcase with hover animations
5. **Education**: Timeline-style education history
6. **Certificates**: Certificate display
7. **Contact**: Contact information and contact form

## File Structure

```
pfbharath/
├── index.html          # Main HTML file
├── css/
│   └── styles.css      # All stylesheets
└── js/
    └── script.js       # JavaScript for animations and interactions
```

## How to Use

1. Open `index.html` in your web browser
2. Customize the content in `index.html` with your personal information
3. Update the styles in `css/styles.css` if needed
4. Modify animations and interactions in `js/script.js`

## Customization

- Replace placeholder text and images with your actual content
- Update colors, fonts, and spacing in the CSS file
- Add your own projects, skills, and certificates
- Modify the typing animation words in `script.js`
